# Résidence Plein Soleil 官網

## 快速部署到 GitHub Pages

1. Fork 或建立新 repository 名稱：`residence-plein-soleil`
2. 上傳 4 個檔案：`index.html`, `style.css`, `lang.js`, `README.md`
3. 前往 Settings → Pages → Source 選擇 `main` 分支
4. 網址：`https://你的帳號.github.io/residence-plein-soleil`

## 自訂化

- 更換 Google Maps：修改 `index.html` 第 90 行 iframe
- 新增房型頁：複製 `index.html` 改名 `rooms.html`
- 上傳圖片：新增 `assets/images/` 資料夾
